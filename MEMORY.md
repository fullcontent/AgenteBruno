# MEMORY.md — memória longa curada

Este arquivo guarda o que precisa sobreviver entre sessões e é pequeno o suficiente para carregar sempre. Não é um log — é curado. Detalhe granular vive em `memory/decisions.md`, `memory/ideas.md`, `memory/lessons.md`, `memory/tasks.md`.

## Princípios de curadoria

- Só entra aqui o que muda a forma de decidir ou agir em qualquer sessão futura, não só na atual.
- Prefira poucas linhas de alta densidade a um histórico completo.
- Quando um fato mudar, atualize ou remova — nunca acumule versões contraditórias.
- Datas relativas ("semana passada") não entram aqui; converta para datas absolutas antes de registrar.
- Ligue memórias relacionadas usando o nome do arquivo/seção, não duplique conteúdo entre root files e `memory/`.
