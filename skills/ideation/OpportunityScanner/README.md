# Opportunity Scanner (Triagem Inicial)

**Objetivo:** antes de investir tempo sintetizando padrões ou desenhando MVP, fazer uma triagem rápida da oportunidade bruta — é um sinal real de mercado/necessidade, ou ruído? Vale entrar na esteira completa?

**Quando usar:** etapa 1 (primeira) da esteira `ValidateNewIdea` — antes do `PatternSynthesizer`. É o filtro de entrada: nem toda ideia mencionada merece as 7 etapas seguintes.

Ver `workflow.md` para o processo.
