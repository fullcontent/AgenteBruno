# Minimal Architect (Less is More)

**Objetivo:** pegar o núcleo de um projeto (MVP) e "cortar a gordura", garantindo uma experiência de usuário (UX/UI) sem fricções, elegante e onde o simples impera.

**Quando usar:** etapa 5 da esteira `ValidateNewIdea` — depois do `MVPSculptor`, antes do `RiskStrategist`.

Ver `workflow.md` para o processo.
