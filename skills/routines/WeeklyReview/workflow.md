# Fluxo de Trabalho (WeeklyReview)

1. **Trigger**: Todo domingo à tarde/noite.
2. **Coleta**:
   - Aciona `OPERACOES`: "Quais as grandes entregas/reuniões críticas da próxima semana?"
   - Aciona `COMUNICACAO`: "Temos threads abandonadas essa semana?"
   - Aciona `FINANCAS`: "Há notas para emitir ou recebimentos para conferir nessa virada de semana?"
   - Aciona `PRODUTO` / `CRIATIVO`: "Baseado em NOW.md, qual deve ser a prioridade #1 do Bruno amanhã?"
3. **Síntese (Orquestrador)**:
   - Cruza as respostas. Elege apenas 1 Movimento Estratégico (One Move) para a semana.
4. **Postagem**:
   - Tópico: `#resumo-diario`.
   - Seções: ⚠️ PENDÊNCIAS, 🗓 PRÓXIMA SEMANA, 💰 FINANCEIRO, ⚡ O MOVIMENTO (ONE MOVE).
